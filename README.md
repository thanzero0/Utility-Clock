# 🕰️ Premium Digital Clock

A beautifully crafted, ultra-minimalist digital clock with a premium glassmorphism aesthetic. Designed for focus and elegance.

## ✨ Features

- **Glassmorphism UI**: Stunning backdrop blur and subtle glow effects.
- **Smooth Animations**: Fluid transitions and interactive elements.
- **Customizable Display**:
  - Toggle between **12-hour** and **24-hour** formats.
  - Show or hide the **dynamic date** display.
  - Choose your precision: **H**, **H:M**, or **H:M:S**.
- **Responsive Design**: Looks perfect on screens of all sizes.
- **Dark Mode Native**: Easy on the eyes, optimized for OLED displays.

## 🛠️ Technology

Built with pure simplicity:
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox, Glassmorphism, and Keyframe animations.
- **JavaScript (ES6+)**: Real-time state management and UI interactions.

## 🚀 Getting Started

Simply open `index.html` in any modern web browser.

```bash
# Clone the repository
git clone https://github.com/yourusername/premium-digital-clock.git

# Open index.html
cd clock
open index.html
```

## ⚙️ Settings

Click the **Gear Icon (⚙️)** in the bottom-left corner to access the settings panel:
- **Show Date**: Toggle the visibility of the current date.
- **24H Format**: Switch between AM/PM and 24-hour time.
- **Precision**: Select how much detail you want to see.
